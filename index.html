Case Study: AI-Powered Market Risk Analysis

Project Overview

An autonomous, serverless workflow I designed and built from scratch to automate market risk analysis. The system fetches daily news headlines, uses Google's Gemini AI to produce a structured risk assessment, and distributes the intelligence to a live Airtable dashboard and a proactive Asana task management system.

The Challenge

Traditional market analysis for a new region is manual, time-consuming, and fails to capture real-time shifts in the business environment.

My Role & System Architecture

As the sole architect, I designed and coded the end-to-end intelligence pipeline using Google Apps Script. The workflow:

  1. Triggers daily to fetch news from pre-configured RSS feeds.
  2. Sends headlines to the Gemini AI API within a carefully engineered prompt to return a risk analysis in a strict JSON format.
  3. Distributes the structured data via API to an Airtable dashboard and an Asana project for proactive alerts.

Final System View

The live Airtable database below is the central hub for viewing the daily, AI-generated risk analysis.

Key Challenges & Resolutions

This project involved significant real-world debugging:

  • API Errors (404s & 403s): Resolved API model rejection and access-denied errors by iteratively testing model versions and re-configuring API token permission scopes.
  • Data Parsing Failures: Implemented robust extraction logic using regex to guarantee a clean JSON string for parsing, overcoming AI response formatting inconsistencies.
